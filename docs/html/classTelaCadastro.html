Descrição detalhada

Representa a tela que permite ao jogador digitar seu nome após atingir uma alta pontuação.

Esta tela é instanciada quando o jogo termina e a pontuação do jogador é alta o suficiente para entrar no ranking. Ela captura a entrada do teclado para formar o nome do jogador e, ao confirmar, usa o PlayerLogger para salvar o novo recorde.
