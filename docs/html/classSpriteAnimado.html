Descrição detalhada

Classe responsável por animar sprites usando uma sprite sheet.

Esta classe gerencia a exibição de sprites animados, manipulando os quadros (frames) individuais de uma imagem que contém múltiplos quadros em sequência. Ideal para animações simples de personagens ou objetos.

A imagem é dividida em numFrames de tamanho igual, com frameLargura x frameAltura, e a animação é exibida a uma taxa de quadros definida por fps.
