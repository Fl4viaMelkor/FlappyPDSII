Descrição detalhada

Implementa uma hitbox retangular.

A posição do retângulo é definida a partir de seu canto inferior esquerdo, e ele possui uma largura (base) e altura fixas. Pode ser utilizado para colisões com obstáculos, personagens ou áreas de interação.
