Descrição detalhada

Representa a tela padrão de "Fim de Jogo".

Esta tela é exibida quando a partida termina e a pontuação do jogador não foi alta o suficiente para entrar no ranking de recordes. Ela mostra a pontuação final e oferece opções para reiniciar, voltar ao menu ou sair do jogo.
