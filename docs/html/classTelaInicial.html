Descrição detalhada

Representa o menu principal do jogo, servindo como ponto de partida.

Esta tela exibe o título do jogo e as opções de navegação (Iniciar Jogo, Recordes, Sair). Ela gerencia a seleção do menu via teclado e exibe uma animação como cursor.
